# 项目概述
欢迎来到 **DocTrans** 测试文件。本文件旨在验证 LLM 在翻译文本时是否能保持格式。

## 验证功能
* **格式**: 这是 *斜体*，这是 **粗体**。
* **列表**:
    1.  第一个项目，带有 [链接到 Google](https://google.com)
    2.  第二个项目，带有 `code_span`
    3.  一个嵌套项目：
        * 深层嵌套的列表项。

> [!提示]
> 这是提示框。它应该在保持 `[!TIP]` 语法不变的情况下进行翻译，以便在 Docusaurus/Obsidian 中使用。
